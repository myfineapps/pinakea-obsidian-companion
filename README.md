# ONA.UNO Companion (Obsidian Plugin)

First-party Obsidian companion plugin for the **ONA.UNO** macOS app.

## What You Get

- **Open in ONA.UNO App** from a note
- **Chat in ONA.UNO App** from a note
- **Connection status indicator** for the current vault

## Install (Beta via BRAT)

1. Install and enable BRAT in Obsidian.
2. In BRAT, choose **Add beta plugin**.
3. Enter `myfineapps/ona-uno-obsidian-companion`.
4. Install/update through BRAT.

## Manual Install (Fallback)

1. Download release assets from:
   - <https://github.com/myfineapps/ona-uno-obsidian-companion/releases>
2. Copy `main.js`, `manifest.json`, and `styles.css` into:
   - `<vault>/.obsidian/plugins/onauno-obsidian-companion/`
3. Enable **ONA.UNO Companion** in Obsidian Community Plugins.

## Requirements

- Obsidian desktop
- ONA.UNO installed on macOS
