# pinakea Companion (Obsidian Plugin)

First-party Obsidian companion plugin for the **pinakea** macOS app.

## What You Get

- **Open in pinakea App** from a note
- **Chat in pinakea App** from a note
- **Connection status indicator** for the current vault

## Install (Beta via BRAT)

1. Install and enable BRAT in Obsidian.
2. In BRAT, choose **Add beta plugin**.
3. Enter `myfineapps/pinakea-obsidian-companion`.
4. Install/update through BRAT.

## Manual Install (Fallback)

1. Download release assets from:
   - <https://github.com/myfineapps/pinakea-obsidian-companion/releases>
2. Copy `main.js`, `manifest.json`, and `styles.css` into:
   - `<vault>/.obsidian/plugins/pinakea-obsidian-companion/`
3. Enable **pinakea Companion** in Obsidian Community Plugins.

## Requirements

- Obsidian desktop
- pinakea installed on macOS
